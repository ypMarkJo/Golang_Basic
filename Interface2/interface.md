인터페이스   
======================
> 인터페이스에 대해 알아보자.

# Object
> 상태+기능 -- 기능   

**기능 == 공개 기능(-> 관계) + 내부 기능**   

Object에서 관계의 종속성을 분리(decoupling)-> 인터페이스
> 관계만 정의되면 오브젝트의 종류는 무관해진다.(확장성이 올라간다.)

ex) 빵과 잼은 바른다라는 관계만 인터페이스로 정의하면 잼의 종류는 무관해진다.

# 선언 방법
```Go
type Jam interface{
    이름 (입력) 출력
    이름 (입력) 출력
    이름 (입력) 출력
}
```
